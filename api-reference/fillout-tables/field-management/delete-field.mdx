---
title: 'Delete Field'
description: 'Delete a field from a base'
---

Delete an existing field from a base. This action cannot be undone, and all data associated with the field will be permanently deleted.

### Base URL
```
DELETE https://api.fillout.com/v1/tables/bases/:baseId/fields/:fieldId
```

### Parameters

<ParamField header path="baseId" type="string" required>
  The ID of the base containing the field
</ParamField>

<ParamField header path="fieldId" type="string" required>
  The ID of the field to delete
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Confirmation of deletion
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl -X DELETE https://api.fillout.com/v1/tables/bases/base_123/fields/fld_abc \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.fillout.com/v1/tables/bases/base_123/fields/fld_abc', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.delete(
    'https://api.fillout.com/v1/tables/bases/base_123/fields/fld_abc',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY'
    }
)

data = response.json()
```
</CodeGroup>

### Example Response

```json
{
  "success": true
}
```

<Warning>
  Deleting a field is permanent and cannot be undone. All data stored in this field will be lost.
</Warning>
