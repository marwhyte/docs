---
title: 'Delete View'
description: 'Delete a view from a base'
---

Delete an existing view from a base. This action cannot be undone, but the underlying data will remain intact.

### Base URL
```
DELETE https://api.fillout.com/v1/tables/bases/:baseId/views/:viewId
```

### Parameters

<ParamField header path="baseId" type="string" required>
  The ID of the base containing the view
</ParamField>

<ParamField header path="viewId" type="string" required>
  The ID of the view to delete
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Confirmation of deletion
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl -X DELETE https://api.fillout.com/v1/tables/bases/base_123/views/view_abc \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.fillout.com/v1/tables/bases/base_123/views/view_abc', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.delete(
    'https://api.fillout.com/v1/tables/bases/base_123/views/view_abc',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY'
    }
)

data = response.json()
```
</CodeGroup>

### Example Response

```json
{
  "success": true
}
```

<Note>
  While deleting a view removes the view configuration, it does not affect the underlying data in your base.
</Note>
